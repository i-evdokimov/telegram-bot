# 🤖 Telegram Bot для сбора заявок

## 📌 Описание
Бот собирает заявки от клиентов через Telegram и сохраняет их в базу данных.

## ⚙️ Функционал
- 📩 Пошаговая анкета (FSM)
- 👤 Сбор имени
- 📱 Сбор телефона
- 💬 Комментарий клиента
- 🗄 Сохранение в PostgreSQL
- 🔔 Уведомление админа

## 🧠 Технологии
- Node.js
- node-telegram-bot-api
- PostgreSQL
- FSM (состояния пользователя)

## 🚀 Как работает
1. Пользователь нажимает "Оставить заявку"
2. Бот задаёт вопросы
3. Данные сохраняются
4. Админ получает уведомление
